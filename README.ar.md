أداة متقدّمة ومفتوحة المصدر للتحكّم بالـ API المخفي لراوترات ZTE الـ 5G. تسوّي كل
اللي تسوّيه صفحة الراوتر الأصلية — وأكثر بكثير — وكل شي **محلي ١٠٠٪**: جهازك يتكلّم
مع راوترك فقط. بدون سحابة. بدون تتبّع.

> **الموديلات المدعومة:** MC801A · MC801A1 · MC888 · MC889 · MC8020 · وغيرها من راوترات ZTE الـ 5G.

## ✨ المميزات

- **📊 الرئيسية والمراقبة الحية** — الموديل، IMEI، الباندات، RSRP/SINR، الحرارة، ورسوم حية كل ثانية.
- **⚡ المُحسِّن التلقائي** — أوضاع بضغطة وحدة:
  - 🚀 **الوضع السريع** — يفك كل شي لأعلى سرعة.
  - 🎮 **القيمرز** — يقيس البنق/التذبذب/الفقد الحقيقي ويقفل على الأقل تأخيرًا.
  - ⚖️ **البلنس** — أقوى وأثبت اتصال.
  - 📶 **الشبكة** — تبديل 4G / 5G.
- **🔒 أدوات LTE / 5G** — قفل الباند، قفل الخلية، وتجميع باندات 5G (مثل `n41 + n78`).
- **🔓 فتح الميزات (Feature Unlock)** — يُظهر أدوات قفل الباند/الخلية اللي تخفيها واجهة مزوّد الخدمة. **قائم على الاكتشاف، بدون أي قيم ثابتة:** التطبيق يقرأ فيرموير راوترك نفسه ويعرض التحكّم فقط إذا كان الأمر موجودًا فعلًا على جهازك — وإلا يكتب *«غير متوفر على هذا الموديل»* بدل زر ميت. وكل قفل مربوط بالوضع الآمن (إرجاع تلقائي) وزر 🚨 الاستعادة.
- **🛟 الوضع الآمن** — يرجّع تلقائي لو انقطع الاتصال ٦٠ ثانية بعد قفل، وزر 🚨 **استعادة** بضغطة.
- **🧠 مستكشف/وحدة الـ API + وضع المطوّر** — يكتشف أوامر الراوتر المخفية تلقائيًا.

## 🤖 بناء APK الأندرويد (سحابي — المفضّل)

1. ارفع المشروع على مستودع **GitHub** (خاص عادي).
2. افتح المستودع → تبويب **Actions** → **Build Android APK** → **Run workflow**.
3. انتظر ~٥ دقايق → افتح التشغيل بعد ما يخلص → نزّل ملف **`ZTE-Router-Manager-apk`** → فك الضغط → `app-debug.apk`.
4. ثبّته على جوالك (فعّل «تثبيت من مصادر غير معروفة») ووصّل الجوال بـ **واي‑فاي الراوتر**.

## 🚀 البداية السريعة

1. افتح التطبيق → اكتب **باسورد الراوتر** → علّم **تذكّرني** → **دخول**.
2. لأقصى سرعة: **المُحسِّن → 🚀 الوضع السريع**.
3. لإجبار 5G السريع: **5G / NR → preset `n41 + n78 (CA)` → Lock**.
4. لو صار خلل: **🚨 استعادة** (الشريط العلوي) يرجّع كل شي تلقائي.
